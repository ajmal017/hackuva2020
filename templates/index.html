StockPredictor

What is StockPredictor?

StockPredictor predicts future stock movement using machine learning and accounts for recent events such as the coronavirus. There are two services StockPredictor currently offers.

1. MLR Model - A Multiple Linear Regression Model that can predict whether a given stock will increase or decrease over the course of the day trained on data from the past 60 days. Specifically, this model is trained on the following features over time: Technical Indicators (RSI, MACD, Bollinger Bands, SMA), News Sentiment (Sentiment score and magnitude analyzed from compiled daily headlines), Coronavirus Statistics (% change in deaths, confirmed, and recovered).

2. AutoML Model - A model trained on 20 years of historical data that predict which way the Dow will move in the next 24 hours. This model was trained using Google's AutoML with previously mentioned technical indicators as features.

StockPredictor was created on Google Cloud in DataLab . It utilizes the Google News API to gather news relating to the stock market and given stocks. The Google Cloud Natural Language API was used to analyze the sentiment of daily news headlines. The Google Cloud AutoML Tables API was used to create and deploy a binary classification model on 20 years of historical data that assesses whether the Dow will move up or down on any given day. All this was done on a virtual instance from the Google Compute Engine. All financial data was accessed using the free Alphavantage API. The web portion of the project was constructed using Flask.

How do I use StockPredictor?

You can either make a POST request using cURl or another requests library to /api/mlr/ and /api/automl_long_term/, or click the get started button below!
